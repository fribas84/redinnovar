# Como correr el proyecto
Tenes que tener instalado pipenv:

- Ubuntu:
```sudo apt install pipenv``` 
- Mac: 
```brew install pipenv``` --> MAC

en la carpeta donde hiciste el clone:

```pipenv install
pipenv shell
python manage.py runserver```
