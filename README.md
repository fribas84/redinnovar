# Como correr el proyecto

https://pipenv-es.readthedocs.io/es/latest/
```sudo apt install pipenv``` --> UBUNTU

```brew install pipenv``` --> MAC

en la carpeta donde hiciste el clone:

```pipenv shell
python manage.py runserver```
